# Market Basket Analysis
Big data mining and processing of transactional data by performing a Market Basket Analysis.

## Set up

#### First time

```bash
python3 -m venv .venv
source ".venv/bin/activate"
pip install -r requirements.txt
jupyter notebook
```

#### Every other time

```bash
source ".venv/bin/activate"
jupyter notebook
```

#### To exit venv use

```bash
deactivate
```
